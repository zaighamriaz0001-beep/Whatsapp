# Whatsapp 

💬 Java Chatting Application
📌 Description

The Java Chatting Application is a desktop-based messaging application developed using Java Swing, Socket Programming. It enables real-time communication between a server and a client over a local network (localhost).

The project demonstrates how networking concepts work in Java while providing a user-friendly graphical interface for sending and receiving messages.

🚀 Features
Real-time client-server communication
Java Swing graphical user interface
Send and receive text messages instantly
Custom chat layout
Message timestamps
Scrollable chat window
Custom icons and profile images
Event-driven programming using ActionListener


🛠 Technologies Used
Java
Java Swing
Socket Programming
AWT
Event Handling
DataInputStream
DataOutputStream
